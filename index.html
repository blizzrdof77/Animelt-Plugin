Animelt

The jQuery plugin for complex animations

Now you can run complex animations in your pages

BETA-VERSION

"Animeltizing", use syntax:

$( sel ).animelt( props,duration,finish );

Other examples:

$("div").animelt({
    width: "40px",
    border: "2px solid #000"
}, 1000, function(){
    console.log("It's finished!");
});

CSS3 Values!

$("div").animelt({
    transform: "rotateX(30deg) rotateY(120deg)"
}, 600);

$(".foo").animelt({
    backgroundImage: "-webkit-linear-gradient(360deg, #000, #fff)"
});
